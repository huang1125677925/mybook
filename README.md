# mybook

个人投资与复盘相关的 Markdown 笔记库：把与 AI 的对话整理成可追溯的问答存档，并把需要长期保留的结论同步到「记忆」目录。

## 仓库结构

| 目录 | 说明 |
|------|------|
| `ai_answers/` | 按日期与时间戳命名的问答记录。每条一般包含 **Q**（问题）与 **A**（回答），便于检索历史分析。 |
| `memories/` | 个人记忆库：持仓、复盘要点等需要反复引用的结构化笔记；子目录（如 `memories/复盘分析/`）用于主题分类。 |

## 使用方式

- 在 `ai_answers/` 中按现有命名习惯新增文件即可保留一次完整对话脉络。
- 将需要固化、精简的结论写入 `memories/`，便于与持仓表、复盘逻辑等长期信息对齐。
