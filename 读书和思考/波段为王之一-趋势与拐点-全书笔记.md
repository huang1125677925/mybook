## 第1章 趋势的力量

**核心：个股走势不是孤立的，要看它在什么大盘环境下运行。**

### 7种运行模式

| 模式 | 特征 | 操作策略 |
|:---|:---|:---|
| ① 大盘引领趋势 | 个股跟着大盘走 | 看大盘做个股 |
| ② 贴近下降趋势上轨 | 弱势股，反弹到上轨就跌 | 逢高减仓 |
| ③ 贴近下降趋势下轨 | 持续阴跌 | 不抄底 |
| ④ 波浪式下降运行 | 跌一段→反弹→再跌 | 等浪形完整才进场 |
| ⑤ 直线下降运行 | 急跌，无反弹 | 绝对不能接飞刀 |
| ⑥ 箱体波动运行 | 横盘震荡 | 高抛低吸 |
| ⑦ 独立于大盘运行 | 走自己的独立行情 | **最强类型，重点关注** |

**核心观点：先判断个股处于哪种运行模式，再决定操作策略，而不是上来就看K线。**

## 第2章 趋势的界定 ⭐本书核心

讲4种工具来**量化界定趋势**，不是凭感觉说"趋势向上"：

### 工具1：直尺
- **最简单的工具**——在K线图上画一条直线
- **上升趋势**：连接两个依次抬高的低点，画上升支撑线
- **下降趋势**：连接两个依次降低的高点，画下降压力线
- 箱体震荡：水平直线

### 工具2：平行通道
- **上升通道**：下轨=支撑线，上轨=平行于下轨的线
- **下降通道**：上轨=压力线，下轨=平行于上轨的线
- 在通道内高抛低吸，突破通道=趋势加速或反转

### 工具3：音叉通道（作者特色工具）
- 以一波行情的起点和终点为基准，画出一个"音叉"形状
- 中轴线=趋势的中心线，上下轨=趋势的边界
- 比平行通道更灵活，能捕捉趋势加速
- 偏离中轴线太远=超买或超卖

### 工具4：波浪
- 用波浪理论（推动浪5波+调整浪3波）界定当前处于什么阶段
- **不追求精确数浪**，而是判断大方向处于哪个阶段

### 核心方法
对上升趋势、下降趋势、箱体震荡这三种走势，分别用这4种工具去界定。

## 第3章 顶部的拐点

用第2章的4种工具 + 30周均线 + 量能，判断**什么时候到顶了**：

| 工具 | 顶部信号 |
|:---|:---|
| 直尺 | 股价跌破上升趋势线 |
| 平行通道 | 股价触及通道上轨+回落 |
| 音叉通道 | 偏离中轴线太远 |
| 30周均线 | 跌破30周均线=中期趋势转弱 |
| 波浪 | 第5浪走完=上涨末端 |
| 量能 | 箱体震荡中放量跌破下轨=向下突破确认 |

## 第5章 跟着趋势走

前面讲了怎么判断趋势和拐点，这一章讲**实际操作**：

- **阻力判断**：用直尺找前期高点连线的位置
- **支撑判断**：用直尺找前期低点连线的位置
- **跟随趋势**：
  - 上升趋势中持股，不要因为"涨多了"就卖
  - 下降趋势中空仓，不要因为"跌多了"就买
- **通道跟随**：在通道内顺势操作，突破通道加仓或减仓
- **波浪跟随**：根据浪形位置调整仓位

**核心观点：趋势是你的朋友，不要跟趋势作对。**

## 第6章 以石攻玉（实战应用）

### ① 量能涨跌的七种形态
- 通过成交量的7种变化模式，辨别主力意图
- 拉升放量、洗盘缩量、出货放量等

### ② 选股策略
- 优先选择**独立于大盘运行**的个股
- 关注量能形态健康的股票

### ③ 时间周期
- 股市运行有内在的时间节律
- 某些特定时间窗口容易产生拐点

### ④ 周期循环对称
- 上涨时间和下跌时间往往存在对称关系
- 上升波段的时间和空间，跟调整波段的时间和空间有一定的比例关系

### ⑤ 波浪预测上证未来
- 用波浪理论对上证指数做中长期预判
