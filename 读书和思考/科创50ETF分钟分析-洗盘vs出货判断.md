## 二、核心案例：下午的急跌为什么是洗盘不是出货

今天下午13:00-13:12这个急跌段，从1.877砸到1.848（-0.48%），之后一路涨回并创了日内新高1.894。这是一次典型的**盘中洗盘**。

### 为什么判断是洗盘？

观察以下信号：

| 信号 | 表现 | 判断 |
|:---|:---|:---:|
| 砸完后能否快速收回 | 13:12最低1.848，13:28就回到1.879 | ✅ 洗盘 |
| 能否创日内新高 | 14:01冲到1.894，创全天最高 | ✅ 洗盘（出货不会创新高） |
| 反弹时是否放量 | 13:28反弹过程放量13万手 | ✅ 真拉升 |
| 砸盘后缩量横盘位置 | 1.848是午后低点，远高于早盘低点1.835 | ✅ 底部抬高，支撑有效 |
| 收盘位置 | 收1.884，远离午后急跌区域 | ✅ 洗盘成功 |

### 出货的相反信号

如果当天走势是以下特征，则更可能是出货：

| 信号 | 出货特征 |
|:---|:---|
| 冲高后放量回落 | 早盘拉到高位，午后持续放量下跌 |
| 创不了新高 | 午后反弹连早盘高点都过不去 |
| 反弹缩量、下跌放量 | 反弹时没有资金跟，砸盘却一直有量 |
| 收盘在低位 | 收在全天最低附近，不拉尾盘 |
| 逐级下台阶 | 低点一个比一个低，没有底部抬高的迹象 |

## 三、洗盘 vs 出货 通用判断框架

### 3.1 日线级别判断

| 维度 | 洗盘 | 出货 |
|:---|:---|:---|
| **价格位置** | 相对低位/上涨中继 | 已有较大涨幅的阶段性高位 |
| **成交量** | 下跌缩量，反弹温和放量 | 下跌放量（主力抛售），反弹缩量（散户接盘） |
| **K线形态** | 长下影线、十字星、小阴小阳 | 大阴线、射击之星、乌云盖顶 |
| **均线** | 回踩MA10/MA20不破，缩量粘合 | 跌破重要均线且反弹无力站回 |
| **资金流向** | 大单净流出不大，甚至尾盘回补 | 超大单持续净流出，小单净流入（散户接盘） |

### 3.2 分钟线级别判断（盘中实时）

| 信号 | 洗盘 | 出货 |
|:---|:---|:---|
| 急跌后的位置 | 砸到支撑位后快速弹起 | 砸下去就站不起来了 |
| 反弹力度 | 反弹放量，量价配合 | 反弹缩量，量价背离 |
| 能否创新高 | 洗完后能创分时新高 | 全天高点越来越低 |
| 底部形态 | 逐级抬高（higher low） | 逐级降低（lower low） |
| 尾盘行为 | 尾盘回拉，收在相对高位 | 尾盘跳水，收在全天低位 |
| 全天振幅 | 振幅大但收红/小跌 | 振幅大且收在低位 |

### 3.3 核心速查口诀

> **砸完能回竞价价上方 → 洗盘**
> **砸完站不回去 → 出货**
>
> **缩量横盘不破低 → 洗盘**
> **横盘放量再破低 → 出货**
>
> **创新高 → 洗盘确认**
> **出不了新高 → 出货嫌疑**
