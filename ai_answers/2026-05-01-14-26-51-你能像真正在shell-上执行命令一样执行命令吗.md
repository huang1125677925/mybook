**Q:** 你能像真正在shell 上执行命令一样执行命令吗

**A:**

是的，我可以通过 **`server_shell_run`** 工具在服务端**真实执行 Shell 命令**，就像你在终端敲命令一样。

---

## ✅ 能力说明

| 能力 | 支持 |
|------|------|
| 执行任意 Shell 命令 | ✅ 真实执行，非模拟 |
| 指定工作目录 | ✅ |
| 设置超时时间 | ✅（可自定义） |
| 返回标准输出/标准错误 | ✅（合并返回） |
| 非交互式命令 | ✅（如脚本、管道、组合命令） |
| **交互式命令**（如 `top`、`vim`、`ssh`） | ❌ 不支持，需 stdin 持续输入 |

---

## ⚠️ 前提条件

该功能需要在服务端设置了环境变量：

```
SERVER_SHELL_TOOL_ENABLED=true
```

如果未设置，调用时会返回未启用的提示。

---

## 🚀 可以帮你做的事

```bash
# 查文件/目录
ls -la /root/test_back_end/

# 看日志尾部
tail -n 100 /path/to/log.log | grep ERROR

# 看系统状态
free -h && df -h && top -bn1 | head -10

# 安装依赖
pip install pandas tushare

# 跑脚本
python3 /path/to/script.py

# 查进程
ps aux | grep python
```
