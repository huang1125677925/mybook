**Q:** realtime list 是什么工具

**A:**

`realtime_list` 是一个**A股实时排名/涨跌幅排行**的爬虫工具，用于抓取当日的实时行情快照。

## 🔧 工具说明

| 项目 | 内容 |
|------|------|
| **工具名** | `tushare_stock_market_realtime_list` |
| **数据来源** | 东方财富等行情源（爬虫抓取） |
| **用途** | 获取**当前交易日**的实时排名榜单，包括涨跌幅、成交量、换手率等 |

## 📊 它能返回什么

调用后可拿到全市场股票在**当前时刻**的实时排名数据，包括但不限于：
- 股票代码/名称
- 最新价
- 涨跌幅（%）
- 涨跌额
- 成交量（手）
- 成交额（元）
- 换手率
- 市盈率（动态）
- 总市值/流通市值
- 量比
- 60日涨跌幅等

## 🎯 典型使用场景

| 场景 | 说明 |
|------|------|
| **实时涨幅榜** | 现在哪些股票涨得最多 |
| **实时跌幅榜** | 现在哪些股票跌得最惨 |
| **成交量排名** | 现在哪些股票成交最活跃 |
| **换手率排名** | 现在哪些股票换手最充分 |
| **快速筛选** | 盘中快速浏览全市场状态 |

## 🔄 与 `rt_k` 的区别

| 工具 | 区别 |
|------|------|
| **`realtime_list`** | ⭐ **全市场截面数据**，一次获取所有股票当前的报价和排名，类似东方财富的"实时行情"页面 |
| **`rt_k`** | **单只股票**的实时日K线（当天开高低收量） |
| **`realtime_quote`** | **单只/少量股票**的实时盘口五档报价 |
| **`realtime_tick`** | **单只股票**的实时逐笔成交明细 |

也就是说，如果你想**一次看全市场今天谁涨得最好、谁跌得最惨**，`realtime_list` 是最合适的工具。而如果要看某只股票的实时买卖五档，需要用 `realtime_quote`。
