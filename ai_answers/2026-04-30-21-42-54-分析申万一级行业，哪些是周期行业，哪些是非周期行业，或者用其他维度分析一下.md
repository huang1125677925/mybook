## 申万一级行业（2021版）周期属性全景分类

### 第一维度：按"经济敏感度+盈利确定性"分三大类

---

### 🔵 一、强周期行业（盈利随宏观周期大幅波动）

这些行业的景气度与 **PMI、PPI、固定资产投资、全球大宗商品价格**高度同步。

| 行业 | 驱动因子 | 周期风格 | 典型特征 |
|------|---------|---------|---------|
| **煤炭** | 能源价格、供给政策 | 早周期→后周期 | 供给侧收紧+能源危机时爆发；红利逻辑在高煤价时成立 |
| **有色金属** | 全球PMI、美元、新能源需求 | 早周期 | 铜铝跟经济，锂钴跟产业趋势；贵金属（金/银）有独立避险逻辑 |
| **基础化工** | PPI、油价、下游需求 | 中周期 | 品种分化大，MDI/钛白粉/氟化工各有独立周期 |
| **钢铁** | 地产基建、供给限产 | 中周期 | 传统"铁公基"映射，弹性大但趋势持续期短 |
| **石油石化** | 油价、炼化价差 | 后周期→全周期 | 上游跟油价，中游跟价差，下游跟消费 |
| **建筑材料** | 地产开工、基建投资 | 中周期 | 水泥有"区域+季节"双重周期，玻璃跟竣工 |
| **基础建设/建筑装饰** | 专项债、地产新开工 | 中周期 | 订单驱动，毛利率低，跟政策预期走 |

---

### 🟢 二、弱周期/防御性行业（盈利相对独立于经济周期）

需求刚性，宏观经济下行时业绩韧性最强。

| 行业 | 防御逻辑 | 适合场景 |
|------|---------|---------|
| **食品饮料** | 必选消费+品牌壁垒 | 弱市避风港，利率下行期大消费受追捧 |
| **医药生物** | 人口老龄化+刚需 | 熊市+震荡市首选；集采扰动下子行业分化 |
| **公用事业** | 水电燃气刚需+高股息 | 利率下行期的"类债"资产，弱市中绝对收益来源 |
| **交通运输** | 物流刚需（快递/高速） | 弱市中相对抗跌，但航空航运有独立周期 |
| **银行** | 高股息+低波动 | 熊市"压舱石"，但隐含净息差下行风险 |
| **纺织服饰** | 必选消费属性 | 弱市中相对稳定，但成长性有限 |
| **农林牧渔** | 猪周期独立于宏观 | 不跟经济走，跟猪价/粮食价格走 |

---

### 🟠 三、介于周期与非周期之间（需细分）

| 行业 | 分类逻辑 | 关键判断维度 |
|------|---------|------------|
| **电力设备** | 新能源产业周期=成长+周期双属性 | 看新能源装机增速、储能政策、硅料/锂价 |
| **电子** | 消费电子靠创新周期，半导体靠国产替代+全球周期 | 看半导体销售额同比、手机出货量、AI资本开支 |
| **计算机** | 靠政策+产业趋势，弱宏观 | AI信创是成长属性，传统IT外包偏防御 |
| **军工** | 国防预算独立于经济周期 | 真正的"逆周期"行业，需辨识订单节奏 |
| **房地产** | 政策周期远大于经济周期 | 看销售/融资/拿地三环节，当前是"政策底→市场底"传导中 |
| **非银金融（券商/保险）** | 券商=牛市旗手（市场成交量驱动），保险=利率敏感+保费周期 | 券商强β，保险偏价值 |
| **机械设备** | 跟制造业资本开支 | 看企业利润→扩产→设备采购，滞后周期 |
| **汽车** | 消费+产业升级双重属性 | 乘用车=可选消费看居民信心，新能源车=成长看渗透率 |
| **家用电器** | 地产后周期+消费升级 | 竣工→装修→买家电，1-2年滞后；白电龙头有消费属性 |
| **传媒** | 内容供给周期+AI催化 | AI+游戏+影视，成长属性渐强，盈利波动大 |
| **通信** | 运营商=公用事业，设备商=科技周期 | 看5G/算力资本开支 |
| **商贸零售/社会服务** | 消费复苏弹性最大 | 旅游/酒店/免税，跟居民消费信心走，弹性大 |

### 📊 更实用的三维分析框架

除了"周期vs非周期"的二分法，推荐用以下三个维度叠加判断：

```
┌─────────────────────────────────────────────┐
│                 行业三维定位                   │
├──────────┬──────────┬──────────┬──────────────┤
│ 维度     │ 强       │ 中       │ 弱           │
├──────────┼──────────┼──────────┼──────────────┤
│ 经济敏感度│ 煤/色/化/钢│ 地产/汽/家电│ 公用/食饮/医药 │
│ 成长性   │ 电子/电新/计│ 军工/机械 │ 银行/公用/交运 │
│ 防御性   │ 食饮/医药/公│ 银行/交运 │ 有色/钢铁/化工 │
└──────────┴──────────┴──────────┴──────────────┘
```

**真正的投资价值在于"错位"——**

- 当市场认为煤炭是"强周期"的时候，它可能正在变成"高股息防御资产"
- 当市场认为医药是"防御行业"的时候，CXO和创新药可能是"高成长进攻资产"
- 当市场认为银行是"稳定器"的时候，存量按揭利率重定价可能带来"盈利下行风险"

### 🎯 不同市场环境下的行业选择速查

| 市场阶段 | 最优行业 | 次优行业 | 回避行业 |
|---------|---------|---------|---------|
| **牛市初期** | 券商、保险 | 计算机、电子 | 公用事业、煤炭 |
| **牛市中期** | 电子、电新、军工 | 有色、化工 | 银行、交运 |
| **牛市末期** | 煤炭、石油石化 | 农林牧渔 | 券商、成长 |
| **熊市初期** | 银行、公用 | 食品饮料 | 有色、钢铁 |
| **熊市中期** | 医药、食饮 | 公用事业 | 地产、周期 |
| **熊市末期** | 地产、建筑 | 银行、保险 | 高估值成长 |
| **震荡市** | 医药、电子 | 家电、汽车 | 强周期大宗 |

---

📌 **一句话总结：没有永恒的"周期"或"防御"标签，同一个行业在不同宏观阶段、不同市场估值下会切换属性。分析行业时，比起贴标签，更重要的是识别当前驱动该行业的核心变量在哪里。**
