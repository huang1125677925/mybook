## 📊 板块轮动分析框架

### 核心理念
> **赚钱的不是买好公司，而是买"当下资金最愿意买的方向"。**

### 五维分析体系（实战用）

| 维度 | 看什么 | Tushare可用工具 |
|:---:|:---|:---|
| ① **价格动量** | 区间涨幅、RPS排名 | `sw_daily` / `index_rps` |
| ② **资金流向** | 主力/北向净流入 | `moneyflow_ind_dc` / `moneyflow_ind_ths` |
| ③ **市场宽度** | 板块内上涨占比 | `industry_ma_breadth` |
| ④ **涨停热度** | 板块内涨停家数 | `limit_cpt_list` / `limit_step` |
| ⑤ **估值位置** | PE/PB历史分位 | `sw_valuation_analysis` |

### 🚦 三段分析时间框架

| 周期 | 核心任务 | 关键信号 |
|:---:|:---|:---|
| **近期**（1周~1月） | 识别当前主线 | 涨幅大+放量+涨停多 = 主线确认 |
| **中期**（1~3月） | 发现轮动拐点 | 领涨板块宽度走弱 + 弱势板块资金流入 = 即将轮动 |
| **长期**（6~12月） | 判断风格切换 | 宏观因子变化 + 产业拐点 = 大级别切换 |

### 🔄 经典轮动模式
```
复苏初期  → 金融/地产/周期
繁荣前期  → 科技/制造/可选消费
繁荣后期  → 消费/医药
衰退期    → 公用事业/红利/黄金
```

### 🎯 实战SOP

**每日扫描（5分钟）：** 看涨幅前5行业 → 看资金流向TOP5 → 看涨停板块分布

**每周诊断（15分钟）：** 拉一周/一月涨幅排名表 → 对比上周谁新上榜/谁退榜 → 检查持仓RPS

**轮动触发检查清单（满足≥2项准备调仓）：**
1. ❓领涨板块涨幅过大（>50%/季）？
2. ❓板块内部分化（龙头滞涨、补涨股活跃）？
3. ❓出现新的政策催化低位板块？
4. ❓资金从高位板块持续流出？

### ⚠️ 核心纪律
- **看持续性**：单日大涨不算，3~5日持续才算
- **看资金验证**：价格涨+主力流出 → 诱多
- **不追高位板块**：右侧等回踩，左侧等确认
- **不频繁换仓**：每波轮动延续2~4周，耐心持有
