## 📊 申万一级行业多维度分类全景

### 一、周期性 vs 非周期性（经典维度）

#### 🔴 强周期行业（与经济周期高度正相关）
| 行业 | 逻辑 |
|------|------|
| **煤炭** | 能源价格随宏观需求剧烈波动 |
| **石油石化** | 原油价格周期 + 需求周期 |
| **有色金属** | 工业金属需求跟随基建/制造周期 |
| **钢铁** | 基建、地产、制造需求驱动 |
| **基础化工** | 下游覆盖地产、汽车、纺织等，周期性传导 |
| **建筑材料** | 地产/基建投资周期直接拉动 |
| **建筑装饰** | 工程开工与宏观投资周期高度同步 |
| **房地产** | 典型的政策+利率+经济周期驱动 |
| **银行** | 信贷周期+利率周期+地产风险敞口 |
| **非银金融** | 股市牛熊周期（券商/保险） |
| **汽车** | 耐用消费品，消费信贷周期敏感 |
| **机械设备** | 资本开支周期，企业扩产意愿 |
| **交通运输** | 货运量与GDP高度相关（航运/航空/物流） |

> 🔵 **共13个强周期行业**

#### 🟢 弱周期/非周期性行业（需求刚性，受经济波动影响小）
| 行业 | 逻辑 |
|------|------|
| **食品饮料** | 必选消费，需求刚性最强 |
| **医药生物** | 刚需+老龄化，独立于经济周期 |
| **公用事业** | 水电气，刚需+价格管制 |
| **农林牧渔** | 农产品需求刚性，猪周期更偏独立 |
| **商贸零售** | 偏必选+超市零售，波动较小 |
| **社会服务** | 部分刚需（教育/医疗），部分偏可选 |
| **美容护理** | 偏消费升级，但粘性较强 |
| **环保** | 政策驱动多于经济周期 |
| **国防军工** | 军费开支独立于经济周期 |
| **通信** | 基础设施属性，需求稳定 |
| **传媒** | 内容消费偏弱周期 |

> 🟢 **共11个非周期/弱周期行业**

#### 🟡 半周期/成长驱动（周期属性弱，更多由产业趋势驱动）
| 行业 | 逻辑 |
|------|------|
| **电子** | 消费电子+半导体，创新周期为主，宏观为辅 |
| **电力设备** | 新能源转型大周期，供给驱动 |
| **计算机** | 数字化/国产替代，产业趋势主导 |
| **家用电器** | 地产后周期+消费升级，界于中间 |
| **纺织服饰** | 可选消费偏弱，但有一定的必选底仓 |
| **轻工制造** | 出口+内需混合，周期属性居中 |
| **综合** | 分散化，无统一属性 |

> 🟡 **共7个偏成长/半周期行业**

---

### 二、产业链位置维度

| 产业链环节 | 行业 |
|-----------|------|
| **上游资源** | 煤炭、石油石化、有色金属、钢铁、基础化工 |
| **中游制造/材料** | 建筑材料、机械设备、电力设备、轻工制造、建筑装饰 |
| **下游消费** | 食品饮料、医药生物、家用电器、纺织服饰、美容护理、汽车、商贸零售、社会服务 |
| **TMT/科技** | 电子、计算机、通信、传媒 |
| **金融地产** | 银行、非银金融、房地产 |
| **基础设施/公共服务** | 公用事业、交通运输、环保、国防军工 |
| **农业** | 农林牧渔 |
| **其他** | 综合 |

---

### 三、投资风格维度（价值/成长/防御）

| 风格 | 行业 | 特征 |
|-----|------|------|
| **深度价值** 🏛️ | 银行、煤炭、石油石化、钢铁、建筑材料 | 低估值、高分红、稳定现金流 |
| **周期价值** 🔄 | 有色金属、基础化工、建筑装饰、房地产、交通运输 | 随周期折价/溢价交易 |
| **稳健成长** 🌱 | 食品饮料、医药生物、家用电器、美容护理 | 稳定的ROE和现金流，消费品牌护城河 |
| **高弹性成长** 🚀 | 电子、计算机、电力设备、国防军工、通信 | 高研发、高估值容忍度、产业趋势驱动 |
| **防御型** 🛡️ | 公用事业、农林牧渔、环保、商贸零售 | 低Beta、弱经济关联 |
| **博弈型** 🎲 | 非银金融（券商）、传媒、社会服务 | 政策敏感、事件驱动 |

---

### 四、政策敏感度维度

| 敏感度 | 行业 |
|-------|------|
| **高（政策主导）** 🔑 | 房地产（调控）、银行（利率/信贷）、国防军工（军费）、环保（碳中和）、电力设备（能源政策）、计算机（信创/国产化） |
| **中** ⚙️ | 煤炭（能源安全）、汽车（新能源补贴）、社会服务（教培/旅游政策）、非银金融（监管/注册制） |
| **低** 🏔️ | 食品饮料、医药生物（集采外）、公用事业、农林牧渔 |

---

### 五、一句话总结

> **如果只看周期/非周期，核心结论是：**
> - **上游资源品 + 金融地产 + 资本品制造 ≈ 强周期**（13个行业）
> - **消费刚需 + 公共服务 + 科技成长 ≈ 弱周期或半周期**（18个行业）
> - **真正的"绝对防御型"行业其实只有几个：食品饮料、医药生物、公用事业、农林牧渔**
