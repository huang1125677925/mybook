**📌 策略框架**
| 方向 | 特点 | 代表板块 | 当前状态 |

| 🔵 科技（右侧） | 高动量、高波动、趋势驱动 | 电子/半导体/计算机 | 强势（电子年+86%），PE偏高，需设止损 |

- 左侧提供安全垫，右侧提供弹性

**⚠️ 实战纪律**
1. **动态再平衡**：科技涨多后主动减仓，左侧跌多后加仓，不守死比例
2. **止损纪律**：右侧科技若趋势破位（跌破关键均线），果断减仓
3. **重视分化**：左侧内部（医药vs食品饮料）也会分化，需要具体看个股或ETF

**右侧交易核心逻辑**

**右侧交易核心逻辑**

**📌 定义**
> 右侧交易 = 趋势确认后才入场/出场，不预测顶底，完全跟随市场趋势。
> 核心信条：**"趋势是朋友，截断亏损，让利润奔跑"**

**⚖️ 左侧 vs 右侧 对比**

| 维度 | 左侧交易 | 右侧交易 |
|:---:|:---|:---|
| 入场点 | 下跌中分批买入（猜底） | 趋势反转/突破后买入（确认） |
| 出场点 | 上涨中分批卖出（猜顶） | 趋势破位后卖出（确认） |
| 核心能力 | 估值定价能力 | 趋势识别与执行力 |
| 最大优势 | 成本低、赔率高 | 胜率高、回撤可控 |
| 最大风险 | 抄底抄在半山腰 | 追高、震荡市反复打脸 |
| 适合性格 | 耐心、逆向、左侧布局型 | 纪律性强、顺势、右侧跟随型 |

**🔄 右侧交易完整流程**

**① 趋势识别（选方向）**
- 用 60日/120日均线 或 趋势线 判断大方向
- 均线多头排列（短>中>长）→ 上升趋势
- 指数/板块共振：个股+板块+大盘三者趋势一致时成功率最高

**② 入场信号（选时机）**
- **突破入场**：放量突破前高/平台/趋势线/关键均线
- **回踩确认入场**：突破后第一次回踩均线/支撑位不破，缩量企稳时入场（更优）
- 成交量验证：突破时放量，回踩时缩量
- RPS/动量指标辅助：个股强度排名靠前（如RPS > 85）

**③ 止损设定（控制风险）**
- **固定比例止损**：入场价的 -7% ~ -10%
- **均线止损**：跌破20日/60日均线（趋势线）
- **前低止损**：跌破近期波段低点
- ❗ 核心原则：**入场前先想好止损位，没有止损不进场**

**④ 持仓管理（让利润奔跑）**
- **右侧加仓**：只在浮盈状态下加仓（绝不摊平亏损头寸）
- 分批加仓：突破加1次，回踩确认加1次
- 趋势延续则持有不动

**⑤ 出场信号（锁定利润）**
- **趋势破位**：跌破上升趋势线/关键均线
- **高位放量滞涨**：量能放大但价格不涨（潜在顶部信号）
- **板块/指数共振转弱**：龙头板块转弱，市场情绪退潮
- 分批出场：先减半仓，确认破位再清仓

**📊 右侧交易适合的市场环境**

| 适合 ✅ | 不适合 ❌ |
|:---|:---|
| 强趋势行情（单边上涨/下跌） | 横盘震荡市（反复假突破） |
| 成交量活跃、流动性好 | 缩量盘整、无量阴跌 |
| 板块轮动明显 | 热点散乱、无序轮动 |
| 有明确龙头板块/个股 | 无主线、一日游行情 |

**⚠️ 右侧交易常见陷阱**
1. **追高被套**：突破当天追在高位，次日回调即止损 —— 解决方案：等回踩确认
2. **震荡市反复打脸**：每次突破都是假突破 —— 解决方案：降低频率，等大级别趋势
3. **该止不止**：趋势已破位但舍不得卖，变成"右侧入场、左侧持有" —— 解决方案：设好止损无条件执行
4. **浮亏加仓**：入场后下跌，想着"摊低成本" —— 这是左侧思维，与右侧矛盾

**🎯 一句话总结**
> **"右侧交易的本质不是'追涨杀跌'，而是'确认趋势后才行动，趋势结束就离场'——用止损控制亏损，用趋势捕捉利润，不猜顶不抄底。"**

**📎 与当前布局策略的关系**
当前策略为"左侧（消费医疗）+右侧（科技）均衡配置"，其中科技方向严格执行右侧逻辑：
- 趋势在则持有，趋势破则减仓
- 不止损的右侧不是右侧，是对趋势的幻想
- 科技右侧的红利低波底仓，恰好为右侧提供"容错空间"——科技破位止损后，底仓仍在，心态不崩

## 板块轮动分析框架

### 🔍 核心理念
> 板块轮动 = 资金在不同行业/主题间周期性迁徙。**赚钱的不是买好公司，而是买"当下资金最愿意买的方向"。**

### 📊 五维分析体系

| 维度 | 核心指标 | 数据来源（Tushare工具） |
|:---:|:---|:---|
| ① 价格动量 | 区间涨幅、RPS强度排名 | sw_daily / index_rps |
| ② 资金流向 | 主力/北向净流入、板块资金流 | moneyflow_ind_dc / moneyflow_ind_ths |
| ③ 市场宽度 | 板块内上涨占比、均线之上占比 | industry_ma_breadth |
| ④ 涨停热度 | 板块内涨停家数、连板梯队 | limit_cpt_list / limit_step |
| ⑤ 估值位置 | PE/PB历史分位 | sw_valuation_analysis |

### 🚦 三时段分析框架

**▶ 近期（1周~1月）—— 看谁在加速**
- 拉取申万一级行业近5/20日涨幅排名 → 识别领涨板块
- 检查领涨板块的成交量是否持续放大 → 判断热度持续性
- 查看涨停分布：涨停最多概念板块 → 短线资金聚焦方向
- 信号：**涨得快+放量+涨停多 = 当前主线**

**▶ 中期（1~3月）—— 看轮动路径**
- 对比"本轮领涨" vs "前期领涨" vs "前期弱势"的转换关系
- 观察资金流向是否从高位板块流向低位板块
- 检查市场宽度：领涨板块内部是否出现"指数涨但个股普涨比例下降"（背离信号）
- 信号：**前期领涨板块的宽度走弱 + 前期弱势板块开始有资金流入 = 轮动即将发生**

**▶ 长期（6~12月）—— 看风格切换**
- 成长 vs 价值：科技/半导体 vs 消费/金融 的相对强弱
- 大小盘：大盘指数 vs 小盘指数的相对表现
- 行业基本面（财报增速）驱动：景气度拐点往往先于股价反映
- 宏观因子：利率上行期价值占优，利率下行期成长占优
- 信号：**宏观因子变化 + 产业趋势拐点 = 大级别风格切换**

### 🔄 经典轮动模式

```
复苏初期  →  金融/地产/周期（政策敏感型）
繁荣前期  →  科技/制造/可选消费（成长型）
繁荣后期  →  消费/医药（防御成长型）
衰退期    →  公用事业/红利/黄金（纯防御型）
```

### 🎯 实战操作SOP（标准作业流程）

**Step 1 — 每日扫描（5分钟）**
- 查看今日涨幅前5的申万行业 → 确认主线
- 查看今日板块资金流向TOP5 → 验证资金认可
- 查看涨停板块分布 → 确认打板方向

**Step 2 — 每周诊断（15分钟）**
- 拉取申万一级行业近1周/1月涨幅排名表
- 对比上周排名：哪些行业新上榜（新轮动方向），哪些行业退榜（退潮）
- 检查持仓板块的RPS是否持续 > 85

**Step 3 — 轮动触发检查清单**
- ❓领涨板块是否已经涨幅过大（>50%/季）？
- ❓领涨板块内部是否出现分化（龙头滞涨、补涨股活跃）？
- ❓是否出现新的政策/事件催化某个低位板块？
- ❓资金是否从高位板块持续流出？
- ☑ 满足2个以上 → 准备调仓；满足3个以上 → 果断行动

### ⚠️ 常见误区

1. **看涨跌幅不看持续性**：单日大涨可能是消息刺激，需看3~5日持续性
2. **只看价格不看资金**：价格涨但主力资金净流出 → 诱多概率大
3. **忽视板块内部分化**：行业指数涨不代表所有个股都涨，选龙头不选跟风
4. **追涨已到高位的板块**：右侧不追高，等回踩；左侧不抄底，等确认
5. **轮动太快频繁换仓**：高频切换=手续费+打脸，每波轮动至少延续2~4周

### 🔗 与当前策略的关系

当前框架为"左侧（消费医疗）+右侧（科技）+底仓（红利低波）"：
- **科技板块**：严格执行轮动分析，RPS>85持有，RPS跌破80+资金流出→减仓
- **消费医疗**：等待轮动信号——当科技宽度走弱、消费医药资金开始回流时加仓
- **红利低波**：不参与轮动，常年持有做底仓，仅在极端高估时微调

> **板块轮动的本质不是预测，而是发现——不猜下一个热点是什么，而是等市场告诉你资金正在往哪去，然后跟上。**
