## 申万一级行业（2021版，31个）周期属性与行情规律

### 一、行业分类框架

#### 1. 强周期行业（与经济周期高度正相关，弹性大）
| 行业 | 驱动逻辑 | 行情特征 |
|------|----------|----------|
| **煤炭** | 能源价格、供给侧、冬夏用电高峰 | 通胀受益，经济复苏初期先行，油价联动 |
| **石油石化** | 油价周期、全球供需 | 跟随大宗商品周期，地缘政治催化 |
| **基础化工** | 原材料价格、产能周期 | 商品属性强，PPI上行期领涨 |
| **钢铁** | 基建地产投资、产能约束 | 经济刺激政策受益，供给侧改革弹性大 |
| **有色金属** | 全球工业需求、新能源金属 | 货币宽松利好铜铝黄金，锂钴跟随产业周期 |
| **建筑材料** | 地产基建开工 | 政策刺激先行指标，开工旺季弹性 |
| **建筑装饰** | 基建投资、地产施工 | 订单驱动，估值极低时防御，行情中期补涨 |

#### 2. 弱周期/消费类（受经济波动影响较小）
| 行业 | 驱动逻辑 | 行情特征 |
|------|----------|----------|
| **食品饮料** | 必选消费、品牌溢价 | 穿越牛熊，弱市避险首选，ROE稳定 |
| **医药生物** | 刚需、人口老龄化、创新驱动 | 防御性强，政策扰动大（集采），创新药有成长属性 |
| **家用电器** | 地产后周期、消费升级 | 地产回暖受益，但家电换新有独立周期 |
| **农林牧渔** | 猪周期、粮食安全 | 独立周期（猪周期），与宏观关联弱 |
| **纺织服饰** | 可选消费 | 经济复苏后期走强，品牌化趋势 |
| **美容护理** | 消费升级、悦己经济 | 高毛利率，弱市中抗跌，消费降级时承压 |
| **商贸零售** | 消费渠道 | 正在被电商重塑，传统零售走弱 |
| **社会服务** | 旅游餐饮酒店 | 疫后修复弹性大，节假日催化，消费信心敏感 |

#### 3. 金融地产类
| 行业 | 驱动逻辑 | 行情特征 |
|------|----------|----------|
| **银行** | 息差、资产质量、信贷投放 | 低估值高股息，防御+大盘稳定器；
  经济复苏中期启动，利率上行期有利 |
| **非银金融** | 市场成交量、利率、政策 | 牛市旗手（券商），保险随利率走；
  行情爆发期弹性巨大，成交量放大先行 |
| **房地产** | 政策周期、销售去化 | 政策驱动型，调控松紧决定方向；
  行情短促凌厉，长期趋势向下 |

#### 4. 成长/科技类（高弹性，看重产业趋势）
| 行业 | 驱动逻辑 | 行情特征 |
|------|----------|----------|
| **电子** | 半导体周期、消费电子创新、AI | 全球科技周期同步，芯片/消费电子有独立景气贝塔 |
| **计算机** | 软件国产化、AI、信创 | 主题驱动，牛市中弹性极大；
  预期先行，业绩兑现慢 |
| **通信** | 5G/6G基建、光模块、算力 | 基建先行→应用后行；
  估值低时安全，催化来时爆发 |
| **传媒** | 游戏、影视、AI应用 | 内容供给周期，监管政策影响大；
  牛市中后段补涨 |
| **电力设备** | 新能源（光伏/风电/锂电/电网） | 产业周期+政策双驱动；
  高成长时估值容忍度高，产能过剩时杀估值 |
| **机械设备** | 制造业投资、自动化 | 跟随制造业景气周期；
  与PMI高度相关 |
| **国防军工** | 国防预算、装备列装 | 独立属性，与宏观关联弱；
  强国逻辑长期看好，事件催化 |
| **汽车** | 新能源车渗透率、出口 | 电动化+智能化双线，跨越周期属性 |

#### 5. 防御/公用类（弱市中避险）
| 行业 | 驱动逻辑 | 行情特征 |
|------|----------|----------|
| **公用事业** | 水电煤气、红利 | 典型防御，弱市资金避风港；
  利率下行期估值提升 |
| **交通运输** | 物流快递、航运、航空 | 航运跟随全球贸易周期；
  高速公路+港口有红利属性 |
| **环保** | 政策驱动、运营资产 | 估值低，政策催化时异动 |
| **综合** | 无鲜明赛道 | 个股行情为主，板块效应弱 |

### 二、行情轮动规律总结

#### ⏳ 一轮完整经济周期中的行业轮动顺序

```
复苏初期 → 周期先行（煤炭/有色/化工/钢铁/建材）
  ↓ 利率低位+PMI回升
金融启动（银行/非银金融） 
  ↓ 信用扩张+成交量放大
消费跟进（食品饮料/医药/家电）
  ↓ 就业改善+收入预期提升
成长爆发（电子/计算机/电力设备/汽车）
  ↓ 流动性充裕+风险偏好提升
周期再通胀（石油石化/煤炭/有色）
  ↓ 过热期商品涨价
防御+红利（公用事业/交通运输/银行高股息）
  ↓ 经济放缓+宽松退出
```

#### 📊 不同市场环境下的行业偏好

| 市场阶段 | 偏好行业 | 逻辑 |
|----------|----------|------|
| **普涨牛市** | 非银金融（券商→保险）、科技（电子/计算机）、有色 | 券商先行放量，科技弹性最大，周期补涨 |
| **震荡市/结构市** | 食品饮料、医药、电力设备、AI主题 | 业绩确定性+产业趋势 |
| **弱市/熊市** | 银行、公用事业、煤炭、交通运输 | 高股息防御，低估值避险 |
| **经济强复苏** | 钢铁、化工、有色、建材、建筑 | 投资拉动，PPI上行 |
| **利率下行** | 公用事业、红利指数、长久期资产 | DCF模型分母变小，估值提升 |
| **利率上行** | 银行、保险、能源 | 息差改善，通胀受益 |
| **地缘风险** | 军工、黄金（有色）、能源 | 避险+军备+通胀涨价 |
| **政策催化** | 信创/国产替代、半导体、新能源 | 国家战略，不受经济周期约束 |

#### 🔑 核心口诀
- **牛市买券商，熊市买银行** 
- **复苏买周期，过热买消费**
- **衰退买公用，滞涨买能源**
- **震荡买医药，爆发买科技**

### 三、关键指标映射

| 指标方向 | 利好行业 | 利空行业 |
|----------|----------|----------|
| PMI上行 | 周期（钢铁/化工/有色）+ 机械 | 公用事业（相对） |
| PPI上行 | 煤炭/石油石化/化工/有色 | 中游制造（成本压力） |
| 利率下行 | 成长/科技/公用事业 | 银行（息差缩窄） |
| 社融扩张 | 银行/地产/建筑 | 无（普适度好） |
| CPI上行 | 食品饮料/农林牧渔/商贸 | 可选消费（抑制） |
| 成交量放大 | 券商/金融科技 | 红利类（资金分流） |

### 四、风险提示
- 行业分类并非绝对，传统周期行业中的龙头公司可能已是成长股（如万华化学）
- 政策扰动可能打破周期规律（如供给侧改革、双碳政策）
- 新兴产业（AI/新能源）可能独立于传统周期走出自己的贝塔
- 此框架适用于**中长期配置**，短期交易要看资金面和技术面
