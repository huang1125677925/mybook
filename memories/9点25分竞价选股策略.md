# 9:25 竞价选股策略 Prompt（每日执行模板）

_最后更新：2026-05-07_

---

## 📌 使用说明

> 每天 9:25 集合竞价数据出来后，把这个 Prompt 丢给我，我就会自动执行**竞价筛选 → 连板判断 → 板块热度 → 打分评级 → 输出候选池** 的完整流程。

---

## 🧠 Prompt 模板

```
【9:25 竞价选股】帮我执行今日竞价选股。

请按以下步骤操作：

【步骤1】获取今日集合竞价数据
- 调用 stk_auction（当日9:25竞价），获取所有个股的开盘价、竞价成交量、竞价成交额

【步骤2】获取连板数据
- 调用 limit_step（连板天梯），查看昨日已有的连板股票梯队
- 调用 limit_list_d（今日涨停列表），查看今日所有涨停/跌停股票

【步骤3】获取板块热度
- 调用 limit_cpt_list（涨停最强板块），看今日涨停家数最多的概念板块 TOP10

【步骤4】核心分析逻辑
按以下标准筛选候选股：

4.1 候选池范围
- 从【2板股】和【3板股】中筛选（最高标和首板不优先考虑）
- 外加今日竞价量异常大的首板股

4.2 评分维度（每项1~5分）

A. 竞价量能（权重30%）
   - 竞价成交额在流通市值0.5%以上 → 4~5分
   - 竞价成交额在流通市值0.2%~0.5% → 3分
   - 竞价成交额小于流通市值0.1% → 1~2分（排除）

B. 开盘位置（权重20%）
   - 开在+3%~+6%且竞价放量 → 5分（分歧转一致的最佳区间）
   - 开在+1%~+3% → 3~4分（需观察9:30后承接）
   - 开在涨停价（一字板）→ 排除（买不进去）
   - 开在0%~+1% → 1~2分（弱势）
   - 开在跌停附近 → 排除（高位断板信号）

C. 连板位置（权重20%）
   - 2板 → 5分（最佳博弈区）
   - 3板 → 4分（龙头候选）
   - 首板 → 3分（需更多确认）
   - 4板+ → 1~2分（高位接力风险大）

D. 板块热度（权重20%）
   - 所属板块在涨停TOP5中，且板块涨停>20家 → 5分
   - 在TOP10中 → 3~4分
   - 不在TOP10中 → 1分（排除）

E. 竞价形态判断（权重10%）
   - 昨涨停+今日竞价放量高开 → 5分（加速）
   - 昨烂板+今日竞价放量高开 → 4分（弱转强）
   - 昨一字板+今日竞价放量开在+5%左右 → 3分（分歧）
   - 昨涨停+今日竞价缩量高开 → 1分（有出货嫌疑）

4.3 排除清单（有以下任一特征直接淘汰）
   - ❌ 竞价无量（成交额<流通市值0.1%）
   - ❌ 开盘价低于昨收盘
   - ❌ 所属板块不在涨停板块TOP10
   - ❌ 连续4板以上
   - ❌ ST股票
   - ❌ 近3日有重大利空公告

【步骤5】输出最终候选池

请按以下格式输出：

---

- 数据范围仅限于今天
```

## ⚡ 精简版（快速使用）

如果你不想每次贴大段，可以用这个精简版：

```
9:25竞价选股，帮我跑一遍完整流程：
1. 拉今日竞价数据
2. 拉连板天梯
3. 拉涨停板块TOP10
4. 按5项评分标准（竞价量/开盘位置/连板位置/板块热度/竞价形态）打分
5. 输出候选池（首选+观察+排除）
```

## 📎 配合使用的工具接口

| 接口 | 用途 | 调用时间 |
|:----|:-----|:--------|
| `stk_auction` | 当日9:25实时竞价 | 9:25~9:30 |
| `limit_step` | 连板天梯（昨日的连板股） | 9:25后 |
| `limit_list_d` | 今日涨停榜单 | 盘中+盘后 |
| `limit_cpt_list` | 涨停板块排行 | 盘中+盘后 |
| `dc_hot` / `ths_hot` | 热度辅助验证 | 可选 |
| `moneyflow_mkt_dc` | 大盘资金流向（辅助判断当日情绪） | 可选 |

---

> **核心纪律**：9:25竞价数据只有5分钟窗口（9:25~9:30），建议9:25一过就发Prompt给我，我一般在30秒内返回结果，你还有时间在开盘前看完候选池。
