# 9:25 竞价选股策略 Prompt（每日执行模板）_v2

_最后更新：2026-05-08 | 更新内容：新增风控维度（封板质量/资金流向/累计涨幅/市值过滤），基于中嘉博创(000889)天量炸板badcase修复_

## 📌 使用说明

> 每天 9:25 集合竞价数据出来后，把这个 Prompt 丢给我，我就会自动执行**竞价筛选 → 连板判断 → 板块热度 → 风控校验 → 打分评级 → 输出候选池** 的完整流程。

---

## 🧠 Prompt 模板

```
【9:25 竞价选股】帮我执行今日竞价选股。

请按以下步骤操作：

【步骤1】获取今日集合竞价数据
- 调用 stk_auction（当日9:25竞价），获取所有个股的开盘价、竞价成交量、竞价成交额

【步骤2】获取连板数据
- 调用 limit_step（连板天梯），查看昨日已有的连板股票梯队
- 调用 limit_list_d（今日涨停列表），查看今日所有涨停/跌停股票

【步骤3】获取板块热度
- 调用 limit_cpt_list（涨停最强板块），看今日涨停家数最多的概念板块 TOP10

【步骤4】前置风控校验（新增 —— Badcase防御体系）
★ 注意：以下检查要先于评分执行，任一条件触发需标记风险，严重者直接排除

4.0.1 流通市值检查
   - 调用 daily_basic 获取前一日个股总市值
   - 流通市值 < 30亿 → 直接排除（超小盘股，易被操控，砸盘风险极高）
   - 30亿 ≤ 流通市值 < 60亿 → 扣1分（小盘股，需警惕）
   - 60亿 ≤ 流通市值 ≤ 300亿 → 正常评分
   - 流通市值 > 500亿 → 扣1分（大盘股连板难度大）

4.0.2 前日封板质量检查（核心风控 —— 中嘉博创badcase根源）
   - 调用 limit_list_d 查看昨日该股的封板数据
   - 计算【封单金额 / 前日成交额】比例：
     · < 10% → 直接排除（虚封板，多头力量极弱，极易开板）
     · 10%~30% → 扣2分（封板偏弱，竞价高开有出货嫌疑）
     · 30%~50% → 扣1分（封板一般）
     · > 50% → 正常（封板质量较好）
   - 如果昨日涨停板在14:30后封板（尾盘偷板）→ 扣1分
   - 如果昨日反复炸板（炸板次数≥2次）→ 扣2分

4.0.3 昨日资金流向检查（新增 —— 验证主力真实意图）
   - 调用 moneyflow 查询昨日个股资金流向
   - 昨日特大单（>100万）净卖出 > 0 → 扣2分（涨停板上主力已在出货）
   - 昨日特大单净买入但 < 成交额5% → 扣1分（主力参与度低）
   - 昨日特大单净买入 > 成交额10% → 加1分（主力强做多）

4.0.4 累计涨幅风控（新增 —— 防止高位接盘）
   - 计算该股从首板前低点（或近20日最低收盘价）到昨日的累计涨幅
   - 累计涨幅 > 50% → 直接排除（获利盘太重，高位接力风险极大）
   - 30% < 累计涨幅 ≤ 50% → 扣2分
   - 15% < 累计涨幅 ≤ 30% → 扣1分
   - ≤ 15% → 正常

【步骤5】核心分析逻辑
按以下标准筛选候选股：

5.1 候选池范围
- 从【2板股】和【3板股】中筛选（最高标和首板不优先考虑）
- 外加今日竞价量异常大的首板股

5.2 评分维度（每项1~5分）

A. 竞价量能（权重25%）
   - 竞价成交额在流通市值0.5%以上 → 4~5分
   - 竞价成交额在流通市值0.2%~0.5% → 3分
   - 竞价成交额小于流通市值0.1% → 1~2分（排除）
   ★ 交叉验证：如果前日封板质量得分低（4.0.2扣分≥2），即使竞价量能大，也要降1~2分（可能是出货型放量）

B. 开盘位置（权重20%）
   - 开在+3%~+6%且竞价放量 → 5分（分歧转一致的最佳区间）
   - 开在+1%~+3% → 3~4分（需观察9:30后承接）
   - 开在涨停价（一字板）→ 排除（买不进去）
   - 开在0%~+1% → 1~2分（弱势）
   - 开在跌停附近 → 排除（高位断板信号）

C. 连板位置（权重15%）
   - 2板 → 5分（最佳博弈区）
   - 3板 → 3分（中嘉博创badcase：3板接力风险已显著上升，降1分）
   - 首板 → 3分（需更多确认）
   - 4板+ → 1分（高位接力风险极大，直接降档）

D. 板块热度（权重20%）
   - 所属板块在涨停TOP5中，且板块涨停>20家 → 5分
   - 在TOP10中 → 3~4分
   - 不在TOP10中 → 1分（排除）

E. 竞价形态判断（权重20%）
   - 昨缩量板（一字/秒板）+ 今日竞价放量高开 → 5分（分歧转一致）
   - 昨换手板（封板良好）+ 今日竞价放量高开 → 5分（加速）
   - 昨烂板（反复炸板）+ 今日竞价放量高开 → 3分（弱转强，仍需验证）
   - 昨一字板 + 今日竞价放量开在+5%左右 → 3分（分歧）
   - 昨涨停 + 今日竞价缩量高开 → 1分（有出货嫌疑）

5.3 排除清单（有以下任一特征直接淘汰）
   - ❌ 竞价无量（成交额<流通市值0.1%）
   - ❌ 开盘价低于昨收盘
   - ❌ 所属板块不在涨停板块TOP10
   - ❌ 连续4板以上
   - ❌ ST股票
   - ❌ 近3日有重大利空公告
   - ❌ 流通市值 < 30亿
   - ❌ 累计涨幅（近20日低点起算）> 50%
   - ❌ 前日封单金额 / 前日成交额 < 10%（虚封板）

【步骤6】输出最终候选池

输出格式（评分处同时标注风控扣分项）：

---
```

## 🎯 v1 → v2 核心改动总结

### 新增风控维度（4.0.1 ~ 4.0.4）

| 新增项 | 针对中嘉博创的漏洞 | 效果 |
|:------|:-----------------|:----|
| **流通市值过滤**<br>（<30亿排除，<60亿扣分） | 74亿小盘被砸 | 小盘股降分，超小盘排除 |
| **前日封板质量**<br>（封单/成交额比例） | 二板封单仅591万（虚封） | ✅ **直接排除了中嘉博创这种弱封板** |
| **昨日资金流向验证** | 主力涨停板上已净卖出1.33亿 | 识别"假涨停真出货" |
| **累计涨幅风控**<br>（>50%排除，>30%扣分） | 从3.26→5.35涨64% | ✅ **直接排除了高位接力** |

### 评分权重调整

| 维度 | v1权重 | v2权重 | 原因 |
|:----|:------:|:------:|:----|
| A. 竞价量能 | 30% | **25%** | 单一指标权重过高，容易被竞价放量迷惑 |
| C. 连板位置 | 20% | **15%** | 3板股评分从4→3分，降低高位吸引力 |
| E. 竞价形态 | 10% | **20%** | 新增对前日封板质量的描述，提高形态识别权重 |

## 📎 配合使用的工具接口

| 接口 | 用途 | 调用时间 |
|:----|:-----|:--------|
| `stk_auction` | 当日9:25实时竞价 | 9:25~9:30 |
| `limit_step` | 连板天梯（昨日的连板股） | 9:25后 |
| `limit_list_d` | 今日涨停榜单 + **昨日封板质量数据** | 9:25后 |
| `limit_cpt_list` | 涨停板块排行 | 9:25后 |
| `daily_basic` | **前日流通市值（新增）** | 9:25后 |
| `moneyflow` | **前日资金流向/特大单净买卖（新增）** | 9:25后 |
| `get_daily_data` | **近20日行情，计算累计涨幅（新增）** | 9:25后 |
| `dc_hot` / `ths_hot` | 热度辅助验证 | 可选 |

---

> **核心纪律**：9:25竞价数据只有5分钟窗口（9:25~9:30），新版增加了4个前置风控查询，需要多花约15~20秒，但能显著降低"竞价骗线"踩坑的概率。
