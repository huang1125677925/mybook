## 一、背景说明

"5日t位法"并非标准金融术语，而是雪球社区中对**以5日均线为核心、结合做T（T+0）交易与仓位管理的综合短线交易体系**的简称。

- **5日** = 5日均线（短期趋势参考线）
- **t位** = "T"（做T交易）+ "位"（仓位/位置管理）
- **法** = 一套可执行的规则体系

## 二、核心逻辑

以5日均线作为短期强弱分界线，根据股价与5日线的相对位置决定买卖操作，配合仓位分层管理和日内做T降低持仓成本。

### 5日线信号系统

| 信号 | 含义 | 操作 |
|:---|:---|:-----|
| 股价**站上**5日线 | 短期强势，多头占优 | 持有/逢低加仓 |
| 股价**回踩**5日线不破 | 支撑有效，买盘有力 | 低吸机会 |
| 股价**跌破**5日线 | 短期走弱，趋势可能转变 | 减仓/止损 |
| 5/10/20/30日线**粘合** | 变盘信号，方向待选择 | 等待，不操作 |
| **三线金叉**（5日线+MACD+成交量） | 强化做多信号 | 可加大仓位 |

## 三、做T操作规则

### 正T（先买后卖）— 震荡市/上升市

```
早盘绿盘回踩5日线 → 买入
盘中冲高无量       → 卖出
差价目标：2~4元/股（视股价而定）
```

### 反T（先卖后买）— 下跌市/开盘急拉

```
早盘高开无爆量 → 先卖出部分仓位
尾盘2:30~3:00  → 观察分时双底后买回
```

### 量价背离做T法

```
股价跌 + 放量 → 低吸信号
股价涨 + 缩量 → 高抛信号
```

## 四、仓位管理体系

### 分层配置结构

```
重仓（底仓）   50%   → 不动，作为做T的"本钱"
卫星仓（T仓）  30%   → 单次操作不超过底仓的30%
现金储备       20%   → 应对突发波动
```

### 基于5日线的仓位调整

| 市场状态 | 建议仓位 | 操作方向 |
|:---|:---:|:-----|
| 指数站上5日线 + 未破 | 5~7成 | 择机做差价 |
| 指数在5日线上但震荡 | 3~5成 | 短线谨慎 |
| 指数跌破5日线 | ≤3成或空仓 | 防守为主 |
| 5日观察窗口期内 | 不动 | 不轻易调仓 |

### 风控纪律

| 规则 | 标准 |
|:---|:-----|
| 总仓位上限 | 不超过 **70%** |
| 单只标的占比 | 不超过 **30%** |
| 止损触发 | 跌破5日线 + 次日无法收回 → 果断走 |
| 极端止损 | 亏损超 **25%** + 基本面恶化 → 无条件止损 |
| 情绪化操作 | 禁止凭感觉频繁做T，需有信号依据 |

## 五、入场强化信号（三线金叉）

提高胜率的过滤条件，三个信号同时出现才加大仓位：

1. **5日线拐头向上** ✅
2. **MACD金叉** ✅
3. **成交量放大** ✅

> 三线同时金叉 = 过滤假突破，胜率更高

## 六、适用场景

### ✅ 适合
- 波动性较强的科技股（AI、存储、半导体、光芯片）
- 流动性好的宽基ETF
- 震荡市或温和上升市

### ❌ 不适合
- 无量阴跌的个股
- 一字板/秒板股（无做T空间）
- 极端单边下跌行情

## 七、常见陷阱

1. **主力假震荡** — 主力故意制造分时波动诱使散户频繁做T，最终错失主升浪
2. **逆势频繁做T** — 单边下跌市中做正T = 接飞刀
3. **满仓做T** — 没有底仓和现金的区分，一旦做错没有补救余地
4. **无信号操作** — 凭感觉而不是技术信号做T
