## 二、如何用融资数据判断行情冷热

### 📐 核心框架：融资五维温度计

```
融资数据 → 五个维度 → 冷/暖/热/沸 → 仓位参考
```

### 维度一：融资余额绝对值与历史分位

| 区间 | 温度 | 含义 | 参考操作 |
|:---:|:---:|:---|---:|
| 历史10%分位以下 | ❄️ **极冷** | 市场无人问津，恐慌极致 | 分批布局 |
| 10%~40%分位 | ❄️ 偏冷 | 观望情绪浓厚，有安全垫 | 逐步加仓 |
| 40%~60%分位 | 🌤️ 适中 | 正常市场，融资不极端 | 中性仓位 |
| 60%~85%分位 | 🔥 偏热 | 情绪升温，但未疯狂 | 谨慎做多 |
| 85%分位以上 | 🔥🔥 **过热** | 杠杆累积高位，风险积聚 | 减仓防御 |

**当前**: 28,620亿 → **🔥 过热区间**。A股历史上融资余额超过2.8万亿的时段（2015年、2021年9月）都对应阶段性顶部区域。

### 维度二：融资占比（融资余额/流通市值）

这是**最重要的单一指标**。

| 占比 | 温度 | 含义 |
|:--:|:---:|---|
| < 2.0% | ❄️ 极冷 | 杠杆出清完毕，底部区域 |
| 2.0% ~ 2.5% | ❄️ 偏冷 | 有上行空间 |
| 2.5% ~ 3.0% | 🌤️ 适中 | 健康水平 |
| **3.0% ~ 3.5%** | 🔥 **警戒线** | 需要警惕 |
| 3.5% ~ 4.0% | 🔥🔥 高危 | 过热，容易出现踩踏 |
| > 4.0% | ☠️ **爆雷区** | 2015年水平 |

**当前**: 3.82% → **🔥🔥 高危区间**。这个位置如果再出一根-2%以上大阴线，很容易触发融资盘连锁平仓。

### 维度三：净买入方向（谁在加，谁在跑）

看**前5名行业/个股 + 后5名行业/个股**。

- **前5全部同赛道**（如全是科技）→ 资金集中攻击，抱团极致 → 行情走向高潮但也接近尾声
- **前5分散在不同板块** → 良性轮动，行情可持续
- **后5集中出逃某板块** → 该板块被融资盘抛弃，短期回避
- **热门股融资占比 > 30%** → 该股杠杆过高，一旦回调踩踏严重

**当前**: TOP9全部科技（光模块x3 + 半导体x2 + AIx3 + 存储x1），**抱团极致信号** ⚠️

### 维度四：趋势变化（加速度）

| 形态 | 信号 | 解读 |
|:---:|:---|---:|
| 余额上升 + 净买入递增 | 🟢 加速升温 | 趋势强，可跟随 |
| 余额上升 + 净买入递减 | 🟡 动量衰减 | 上涨末端，谨慎 |
| 余额下降 + 净偿还递增 | 🔴 恐慌出逃 | 可能急跌 |
| 余额下降 + 净偿还递减 | 🟢 恐慌消退 | 底部接近 |

**当前**: 余额连续上升（4/30→5/14 +6.78%），但净买入从5/11的289亿降至5/14的80亿 → **🟡 动量衰减信号**

通俗讲：**增量资金进场速度在放缓**，虽然余额还在创新高，但"新进来的人"越来越少了。

### 维度五：大盘vs融资背离

**融资持续买，大盘开始跌** = 危险背离。

| 场景 | 含义 | 策略 |
|:---|---:|---:|
| 大盘涨 + 融资买 | 正常上涨，资金配合 | 持有 |
| 大盘跌 + 融资卖 | 恐慌出逃 | 减仓 |
| **大盘跌 + 融资仍买** | ⚠️ **融资盘与指数背离** | 往往是调整刚开始，散户在抄底、主力和机构在出货 |
| 大盘涨 + 融资卖 | 行情靠存量或外资推动，非杠杆驱动 | 结构性机会 |

**当前**: 5/13~5/14大盘两连跌，但融资仍净买入+299亿 → **⚠️ 背离信号**。说明散户/活跃资金还在抄底，但指数不跟，这个组合历史上通常意味着调整还没结束。

### 实操建议

1. **不追高融资买入集中的个股** — 中际旭创融资占比21%、天孚通信35%、光迅科技38%，这些票一旦融资盘松动，回调幅度会很大
2. **关注净买入能否止跌回升** — 如果接下来融资净买入继续萎缩到50亿以下甚至转负，确认降温，届时才是更好的入场时机
3. **融资占比回到3.3%以下再重仓** — 当前3.82%的风险收益比不划算
4. **融资数据优先看趋势而非绝对值** — 绝对值高不等于马上跌（可以横住），但"减速"（净买入越来越小）是最危险的信号

## 四、融资数据的局限性

1. **深交所/上交所公布时间** — 融资数据T+1披露（当天数据次日早上可查），**无法用于盘中决策**，只能判断方向
2. **个股融资数据有滞后** — 部分个股融资余额更新到T-2
3. **融资≠主力资金** — 大单资金流向（东方财富/Level2）更能反映主力行为，融资反映的是**活跃散户+游资**的杠杆情绪
4. **牛市中后期融资占比会失真** — 因为股价上涨导致流通市值被动增大，融资占比指标会被"压低"，掩盖真实杠杆水平。更好的做法是看 **融资余额/指数点位** 的相对比值
